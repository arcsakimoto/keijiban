# PDF生成スキル

## いつ使うか（WHEN）
- PDFファイルを作成する時
- 契約書や請求書のテンプレートを作る時

## いつ使わないか（WHEN NOT）
- 編集可能な文書が必要な場合は docx スキルを使用
- データ集計の場合は xlsx スキルを使用

## 手順
1. reportlabライブラリを使用
2. 日本語フォント対応
3. A4サイズ基本
4. 完成後ファイルの内容をコメントで説明
