# プレゼンテーション生成スキル

## いつ使うか（WHEN）
- PowerPointやスライド資料を作成する時
- 提案書やプレゼン資料を作る時

## いつ使わないか（WHEN NOT）
- Word文書の場合は docx スキルを使用
- データ集計の場合は xlsx スキルを使用

## 手順
1. python-pptxライブラリを使用
2. 16:9レイアウト基本
3. 日本語フォント対応
4. 完成後ファイルの内容をコメントで説明
